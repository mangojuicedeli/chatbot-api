# 프로젝트 소개
챗봇 API 만들기

# 프로젝트 환경
- 언어
  - Java

- 프레임워크
  - Spring(boot)

- 사용 기술
  - 빌드 도구: Maven
  - 데이터베이스: H2
  - 기타: JPA (Spring Data JPA)

# 요구 사항 분석
- 구현할 API
  - 시나리오 제작 API: 챗봇의 대화 시나리오를 미리 작성해 두기 위한 API
  - 챗봇 사용 API: 사용자의 입력에 따라 정해진 시나리오대로 답변을 받을 수 있는 메시지 API
  
- 메시지 타입
  - 텍스트, 이미지, 빠른 답변(선택지), 자유입력
  
- 테스트 코드 작성

- 서버 구동 및 코드 실행에 대한 문서화
